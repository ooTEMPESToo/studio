# **App Name**: ModernizeJS

## Core Features:

- Code Input: Input area for pasting code or uploading a zip file of legacy HTML, CSS, and JavaScript project.
- Intelligent React Component Suggestion: Suggest React components based on the structure of the input code, using AI to analyze and determine optimal component divisions; this is achieved by implementing the best AI tool for extracting components.
- CSS to Tailwind Converter: Convert existing CSS classes to Tailwind CSS equivalents.
- Next.js Project Generator: Generate a Next.js project structure based on the converted components and styles.

## Style Guidelines:

- Dark mode background: Dark gray (#121212) to provide a modern and developer-friendly feel.
- Primary color: Vibrant green (#32CD32) to highlight interactive elements and actions.
- Accent color: Electric purple (#8A2BE2) for secondary interactive elements and visual accents.
- Font: 'Inter' (sans-serif) for both headings and body text, providing a clean and modern look.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use sharp, minimalist icons to represent different features and tools.
- Subtle transition animations for code conversion and project generation processes.
