# ModernizeJS

A developer tool to assist in migrating traditional HTML, CSS, and JavaScript projects to modern React, Tailwind CSS, and Next.js.

## Core Features

- **Code Input**: Paste your legacy HTML, CSS, and JavaScript code.
- **Intelligent React Component Suggestion**: Get AI-powered suggestions for breaking your code into React components.
- **CSS to Tailwind Converter**: Automatically convert CSS to Tailwind CSS utility classes.
- **Next.js Project Generator**: Visualize a recommended Next.js project structure.
